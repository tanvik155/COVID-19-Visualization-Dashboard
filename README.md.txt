
# 🦠 COVID-19 Visualization Dashboard  
An interactive COVID-19 data visualization app built using **Streamlit**, fetching real-time data from the **Disease.sh API**.

![COVID-19 Dashboard](https://www.streamlit.io/images/brand/streamlit-mark-color.png)

---

## 🚀 Features  
✅ **Live COVID-19 Data** (Global & Country-wise)  
✅ **Interactive Charts** using Matplotlib, Seaborn & Plotly  
✅ **Sidebar for Country Selection**  
✅ **Top 10 Most Affected Countries**  

---

 🛠️ Installation  
 Step 1: Clone the Repository  

git clone https://github.com/your-username/covid-dashboard.git  
cd covid-dashboard  

Step 2: Install Dependencies    
pip install -r requirements.txt  

Step 3: Run the Streamlit App  
streamlit run app.py  

---

## 📊 Visualizations Included  
📌 **Bar Chart** - COVID-19 cases by country  
📌 **Pie Chart** - Cases vs. Deaths vs. Recoveries  
📌 **Top 10 Affected Countries** - Sorted data  

---

📺 Data Source  
This project fetches real-time COVID-19 data from:  
🔗 [Disease.sh API](https://disease.sh)
